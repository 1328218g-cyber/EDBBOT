대시보드
● 오프라인 연결 대기 중
📘 에디 봇 전체 가이드
이 페이지는 현재 추가된 메뉴와 주요 명령어를 한 번에 확인할 수 있는 통합 안내서입니다.
명령어는 대부분 각 메뉴 안에서 원하는 이름으로 바꿀 수 있으므로, 아래에 적힌 예시는 기본값 기준이며 실제 사용 전 각 메뉴의 입력칸도 함께 확인하는 것을 권장합니다.
1. 기본 사용 순서
토큰 상태 확인 → 입장 설정/자동 접속 구성 → 기능별 메뉴 세팅 → 저장 → 실제 방송 채팅에서 명령어 테스트 순서로 사용하면 가장 안정적입니다.
2. 설정 저장
우측 상단 💾 저장 버튼으로 현재 값을 저장할 수 있으며, 좌측 하단의 설정 내보내기/가져오기로 백업과 복원이 가능합니다.
3. 주의사항
기본값 초기화는 전체 설정을 되돌리는 기능입니다. 적용 전 백업 파일을 먼저 저장해 두는 것이 좋습니다.
🧭 메뉴별 설명
🏠 대시보드
방송 상태, 최근 기록, 요약 정보를 한눈에 보는 첫 화면입니다. 현재 작동 상태를 빠르게 확인할 때 사용합니다.
🚪 입장 설정
입장/자동 메시지 관련 기능을 설정하는 메뉴입니다. 자동 접속 대상, 입장 안내, 반복 메시지 계열을 관리할 때 사용합니다.
⌨️ 단축키 명령어
채팅창에서 ![명령어] 형식으로 빠르게 문구를 출력하는 메뉴입니다. 운영 멘트, 홍보 멘트, 공지 멘트를 짧은 단축키로 줄일 수 있습니다.
👋 지정 인사
특정 시청자 또는 닉네임에 맞춘 개별 인사말을 관리하는 메뉴입니다. 고정 청취자 환영 메시지를 따로 지정할 때 유용합니다.
📋 채팅 로그
수신된 채팅과 시스템 처리 내용을 확인하는 로그 메뉴입니다. 기능 테스트나 문제 원인 확인 시 가장 먼저 확인하면 좋습니다.
🔑 토큰 상태
로그인, 연결, 인증 상태를 점검하는 메뉴입니다. 봇이 정상 동작하지 않을 때 우선 확인해야 하는 영역입니다.
💰 펀딩 관리
펀딩 항목을 만들고, 진행률을 표시하고, 채팅 명령어로 적립/차감을 처리하는 메뉴입니다.
🏁 단비 깃발
깃발 목표치를 만들고 자동 적립 또는 수동 적립을 운영하는 메뉴입니다. 주기 출력과 상태 안내에 적합합니다.
🛡️ 실드 관리
실드 개수를 조회하고 증감하는 메뉴입니다. 방송 운영 중 실드 수량을 빠르게 공유할 수 있습니다.
🎵 신청곡 관리
신청곡 등록, 삭제, 리셋, 접수 마감/재개, 현재곡 안내를 관리하는 메뉴입니다.
🔖 스티커음향
특정 스티커와 효과음을 연결해 반응시키는 기능입니다. 스티커 이벤트에 소리 연출을 더할 때 사용합니다.
🎡 룰렛 설정 / 📊 룰렛 기록
룰렛 항목과 확률, 보상 관리와 기록 조회를 담당합니다. 이벤트형 방송 진행에 적합합니다.
🎲 기타 모듈
주사위, 리액션 타이머, 디데이처럼 보조성 기능을 모아둔 메뉴입니다.
⭐ 애청지수
출석, 복권, 랜킹, 상점 등 참여형 포인트 시스템을 관리하는 메뉴입니다.
🧩 퀴즈 설정
지정 분/초 마다 랜덤으로 퀴즈를 출제하고, 첫 번째 정답자에게 경험치를 지급하는 시스템입니다. 제한시간 초과 시 자동으로 정답이 출력됩니다.
🔊 TTS 설정
TTS 권한 부여 기준, 사용자 목록, 읽기 동작을 관리하는 메뉴입니다.
🖼️ 박제스티커
스티커를 무대나 화면 위에 배치해 연출하는 기능입니다.
⚙️ 메뉴 관리
사이드바 메뉴의 순서와 이미지를 조정하는 보조 메뉴입니다.
⌨️ 주요 명령어 사용법
단축키 명령어
!명령어 같은 트리거를 직접 등록해 응답 문구를 출력합니다. 방송 공지, 후원 안내, 규칙 안내처럼 반복되는 문장을 자동화할 때 사용합니다.
🛡️ 실드 관리
!실드 : 현재 보유 실드 수 조회
!실드 +5 : 실드 5개 추가
!실드 -3 : 실드 3개 차감
DJ/매니저는 증감이 가능하고, 일반 유저는 조회 위주로 사용합니다.
🎵 신청곡 관리
!신청곡 : 현재 대기곡 목록 조회
!신청곡 [가수] [곡명] : 곡 신청
!제거 [번호] : 특정 곡 삭제 (DJ/매니저)
!신청곡 리셋 또는 !리셋 : 목록 초기화
!마감 / !접수 : 신청곡 접수 상태 변경
!우선온 / !우선오프 : 우선 처리 기능 전환
!현재곡 : 현재 재생 중인 곡 안내
🏁 단비 깃발
!깃발 : 현재 깃발 목표/진행 상황 조회
메뉴에서 목표 금액, 자동 적립 여부, 주기 출력 시간을 설정할 수 있으며 필요 시 수동으로도 적립/차감할 수 있습니다.
💰 펀딩 관리
!펀딩 : 진행 중인 펀딩 목록 조회
!펀딩 [번호] [숫자] : 해당 펀딩에 금액 적립 또는 차감
제목, 목표 금액, 출력 포맷, 헤더 문구를 메뉴에서 각각 커스텀할 수 있습니다.
🎲 기타 모듈
!주사위 : 1~6 랜덤 숫자 출력
!리액션 [분] [내용] : 타이머 등록 / !리액션 : 목록 확인
!디데이 [MM-DD] [내용] : 디데이 등록 / !디데이 : 목록 확인
⭐ 애청지수
!내정보 : 내 데이터 조회
!내정보 생성 / !내정보 삭제 : 사용자 데이터 생성 및 삭제
!랭킹 : 순위 조회
!복권 [수량] : 복권 사용
!출석 : 출석 처리
!복권지급 : 복권 지급 (DJ + 지정 권한자)
!상점 : 상점 기능 호출 (DJ + 지정 권한자)
🛠️ 기능별 사용 팁
테스트는 채팅 로그와 함께 확인
새 명령어를 저장한 뒤 실제 방송 채팅에서 입력하고, 채팅 로그 메뉴에서 수신 여부와 응답 여부를 같이 확인하면 오류를 빠르게 찾을 수 있습니다.
권한이 필요한 명령어 주의
실드 증감, 신청곡 삭제/리셋, 접수 상태 변경, 일부 애청지수 기능은 DJ 또는 매니저 전용으로 동작할 수 있습니다.
커스텀 문구 적극 활용
펀딩, 신청곡, 실드, 애청지수, 기타 모듈은 출력 문구를 직접 바꿀 수 있으므로 방송 톤에 맞게 통일하면 완성도가 높아집니다.
백업 습관 권장
메뉴를 많이 커스텀한 뒤에는 설정 내보내기로 JSON 백업본을 남겨 두면 초기화나 재설치 시 복구가 쉽습니다.
